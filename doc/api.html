• To change the format of the response optional ?format=xml or ?format=html, default is html
  • list of cities /city
  • Information about a specific city /city/[city name] where city name matches /[w ]+/
  • Cabs Search /city/[city name]/cabs where city name matches /[w ]+/
    optional ?vehicle_registration_number=[registration part /[w ]+/]
  • Information about a specific cab /city/[city name]/cabs/[vehicle registration number]
    where city name matches /[w ]+/
    where vehicle registration number name matches /[w ]+/
  • Texting interface /city/[City Name]/cabs?MobileNumber=[Mobile Number]&Text=[Car registration to check]
